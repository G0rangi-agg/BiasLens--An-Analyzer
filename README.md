# BiasLens--An-Analyzer
A deep learning-based system that analyzes political news articles to detect media bias through entity-level sentiment and framing analysis.
BiasedStack is an end-to-end deep learning project designed to analyze media framing and detect bias in political news articles. The system goes beyond traditional sentiment analysis by focusing on how specific entities (such as political leaders or parties) are portrayed within context.

## Key Features
 Automated news scraping (real-time + historical data)
 Data cleaning and preprocessing pipeline
 Named Entity Recognition with entity linking
 Aspect-based sentiment analysis (entity-level)
 Bias detection using sentiment + frequency correlation (in progress)
 Planned visualization dashboard (heatmaps, trends, comparisons)
## Tech Stack
  Python
  Pandas
  NLP & Deep Learning Models (LLM-based approach)
  newspaper3k (scraping)
  Regular Expressions + ftfy (data cleaning)
## System Workflow
  Scrape political news articles
  Clean and preprocess textual data
  Identify and unify entities
  Perform contextual sentiment analysis
  Aggregate results to detect bias patterns
